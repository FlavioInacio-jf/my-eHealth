 **[JUnit]** - Usada para testar os métodos das classes.

**[Jfree Chart]** - Usada para gerar os gráficos da aplicação.

**[SQLite jdbc]** - Usada para fazer a conexão com o banco de dados SQLite.

OBSERVAÇÃO IMPORTANTE

Para evitar que os testes sofram interferência de dados já existentes no banco de dados, é necessário que o usuário limpe as tabelas do banco de dados. Isso pode ser feito através das configurações da aplicação, na opção Resetar BD. Pois tem regras de negocios que impedem cadastrar Unidade de saúdes com o mesmo CNPJ e pessoas com o mesmo CPF.
